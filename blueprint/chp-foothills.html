3.1 The Goodstein object and its faithful encoding

Goodstein termination (the mathematical fact). Every Goodstein sequence reaches \(0\): \(\forall m,\ \exists N,\ \texttt{goodsteinSeq}\, m\, N = 0\). Proved in-repo (Dom.goodstein_terminates, the lean-gallery artifact’s proof; mathlib carries the ingredients via ONote.fastGrowing but upstreaming this theorem is a non-goal of this program). It plays two roles here: the faithfulness target the encoding is checked against (Thm 3.6), and the totality witness behind the length function (Def 4.1). A green anchor, not an obligation.

Definition 3.3

Uses

Arithmetic transfinite induction \(\mathrm{TI}({{\lt}}\varepsilon _0)\) (gap). The schema of transfinite induction up to each ordinal below \(\varepsilon _0\), expressed over ONote codes as a family of PA-formulas. This is the object whose provability separates \(\mathsf{PA}\) from \(\mathrm{Con}(\mathsf{PA})\) (Gentzen): PA proves each instance below \(\varepsilon _0\) but not the schema at \(\varepsilon _0\). Net-new nowhere in Lean.

Wainer/Buchholz classification (the deep gap = ordinal analysis of PA). A recursive function is \(\mathsf{PA}\)-provably total iff it is eventually dominated by some \(f_o\) with \(o{\lt}\varepsilon _0\). Its converse half (provably-total \(\Rightarrow \) \({\lt}\varepsilon _0\)-bounded) is proved in the literature by embedding PA into \(\omega \)-logic, applying cut-elimination, and reading off an ordinal assignment \({\lt}\varepsilon _0\) (Buchholz–Wainer 1987). Formalized in no proof assistant; this is the Route-B face of the one intrinsic monument.

  • GoodsteinPA.goodsteinSentence_faithful

Faithfulness bridge (locked anchor). The standard model satisfies the encoded sentence iff every genuine hereditary-base Goodstein sequence reaches \(0\): \(\; (\mathbb {N}\models \gamma )\iff \forall m,\exists N,\ \texttt{goodsteinSeq}\, m\, N = 0\). Proved and kernel-clean (\(\{ \texttt{propext},\texttt{Classical.choice},\texttt{Quot.sound}\} \)). This is what makes a \(\mathsf{PA}\nvdash \gamma \) result about the real Goodstein theorem.
