                Internal max-coefficient iC via course-of-values recursion #

                iC is Rathjen's C (DescentCore.C): C 0 = 0, C (oadd e n r) = max (max (C e) n) (C r). Inside V it recurses on the code value through the subterm bounds (ocExp c, ocTail c < c), so we build it by the same table reduction as ibump: iCTable c = ⟨iC 0,…,iC c⟩, reading the two sub-results out of the table.
