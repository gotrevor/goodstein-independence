            The bridge wsumc (iwseq f n N) i = ipsum f n i #

            The cumulative width of the code (read by BlkRec.wsumc) is exactly the internal partial-sum of iterates ipsum — so BlkRec's elapsed-width identity wsumc (blk m) + off m = m becomes Rathjen's m = psum (blockIdx) + blockOff.
