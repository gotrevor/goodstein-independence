              The ℕ-model of LX parametrised by a set S ⊆ ℕ (lap-13 task (i)) #

              The single most-leveraged lego of the Buchholz Boundedness route: the only member of the M5/M4 instance bundle (GenericZinftyProbe.lean) missing for LX was Structure LX ℕ. We build it here, parametrised by the interpretation S : ℕ → Prop of the set variable X. It doubles as the carrier of Buchholz's truth semantics ⊨^α (there S := {n | |n|_≺ < α}, the ≺-initial segment of norm < α). Also: the two DecidableEq instances (LX.Func k, LX.Rel k) completing the bundle for Finset sequents.

                theorem GoodsteinPA.LangX.eval_Xatom (S : ℕ → Prop) {ξ : Type u_1} (e : Fin 0 → ℕ) (ε : ξ → ℕ) (t : LO.FirstOrder.Semiterm LX ξ 0) :
                (LO.FirstOrder.Semiformula.Eval (structLX S) e ε) (Xatom t) ↔ S (LO.FirstOrder.Semiterm.val (structLX S) e ε t)

                The carrier behaves as Buchholz's ⊨^S. Evaluating the X-atom X t in structLX S yields S of the value of t — i.e. X reads as the set S. This is the defining property that makes structLX S the ⊨^α carrier (take S := {n | |n|_≺ < α}).
