    def GoodsteinPA.base (k : ℕ) :

      @[irreducible]
      def GoodsteinPA.bump (b n : ℕ) :
      ℕ

      Hereditary-base bump. bump b n reads n in hereditary base b and replaces every occurrence of b by b + 1. Peeling the top power (e = log b n, c = n / b^e, r = n % b^e): bump b n = c · (b+1)^(bump b e) + bump b r, with bump b 0 = 0.

      Equations
      • GoodsteinPA.bump b n = if h : n = 0 then 0 else n / b ^ Nat.log b n * (b + 1) ^ GoodsteinPA.bump b (Nat.log b n) + GoodsteinPA.bump b (n % b ^ Nat.log b n)

        def GoodsteinPA.goodsteinSeq (m : ℕ) :
        ℕ → ℕ

        Goodstein sequence seeded at m: goodsteinSeq m k = G k. G 0 = m; G (k+1) bumps the hereditary base k+2 ↦ k+3 in G k and subtracts one (0 a fixed point, as bump b 0 = 0 and 0 - 1 = 0).

        • GoodsteinPA.goodsteinSeq m 0 = m
        • GoodsteinPA.goodsteinSeq m k.succ = GoodsteinPA.bump (GoodsteinPA.base k) (GoodsteinPA.goodsteinSeq m k) - 1
