    X-freeness of a formula (structural; every relation symbol is an ℒₒᵣ-relation Sum.inl) #

    def GoodsteinPA.EmbeddingX.XFreeForm {ξ : Type u_1} {n : ℕ} :
    LO.FirstOrder.Semiformula LangX.LX ξ n → Prop

    A Semiformula LX is X-free when every relation symbol occurring in it is an ℒₒᵣ-relation (Sum.isLeft = true), i.e. none is the set variable X. Defined by structural recursion.
