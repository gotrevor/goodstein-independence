    theorem GoodsteinPA.primrec_natLog :
    Primrec₂ Nat.log

    Nat.log is primitive recursive. Strong recursion: log b n reduces to log b (n/b) and n/b < n whenever 1 < b ≤ n.

    theorem GoodsteinPA.primrec_bump :
    Primrec₂ bump

    bump is primitive recursive. Strong recursion peeling the top power: bump b n calls bump b (Nat.log b n) and bump b (n % b ^ Nat.log b n), both at indices < n (the decreasing_by bounds of Defs.bump), so the value is recoverable from the table of earlier values. Lookups use Option.getD 0; on the genuine table both succeed.
