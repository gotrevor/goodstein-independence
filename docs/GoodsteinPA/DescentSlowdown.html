From the slowed code sequence β with the three structural facts (NF, iCanon (k+1), icmp-descent), the seed m₀ = T̂²_ω(β₀) and bound b k = T̂^{k+2}_ω(βₖ) = ievalNat (k+1) (β k) satisfy the base / inductive-step / positivity preconditions of the Lemma-3.6 run. These are pure code arithmetic — no definability, no X — so the SAME lemma feeds both DescentArith.nonterminating_internal (the 𝚺₁ path, via hbound_of_slowdown) and DescentSemantic.lx_nonterminating (the X-essential path, where β is the X-definable Mlt-descent). The caller supplies b, m₀ and their defining equalities.
