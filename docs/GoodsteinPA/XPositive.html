    def GoodsteinPA.XPositive.XPos {n : ℕ} :
    LO.FirstOrder.Semiformula LangX.LX ℕ n → Prop

    X-positive: no ¬X t subformula — i.e. every negative atom is an ℒₒᵣ-relation (Sum.isLeft), never the set variable X.
